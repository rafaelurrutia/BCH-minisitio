CARACTERÍSTICAS DE LA NUEVA PAGINA DE PERFILAMIENTO

Dependiendo de los roles que le han sido otorgados en cada empresa, el cliente podrá elegir trabajar con una u otra, gracias al “Selector de Empresas” ubicado en la Cabecera de la página.

Podrá navegar por las diferentes secciones de la Plataforma con la barra de Menú o accediendo con un click a las secciones más visitadas a través de los accesos directos. Todos estos accesos dependerán de si el cliente tiene habilitados los permisos y roles necesarios según lo haya establecido su empresa.

Generalidades Perfilamiento Descripción Cabecera

  • Funcionalidades Banconexión 2.0 /
  • Ingreso - Solicitud de Clave

En el nuevo Banconexión 2.0, se mejoró la funcionalidad de Solicitud de Clave, permitiendo su obtención remota. Para hacerlo, el usuario tiene dos opciones:

1 | Solicitud de Clave CON datos de Banconexión (BCX) -Sólo una vez

Solicitud de Clave CON Bcx

Si el usuario quiere obtener su clave de ingreso, y tiene clave de Banconexión, puede utilizarla por una única vez.

2 | Solicitud de Clave SIN datos de Banconexión (BCX).

Solicitud de Clave CON Bcx

Si el usuario quiere obtener su clave de ingreso y NO tiene clave de Banconexión, el sistema le solicitará: rut, teléfono celular y mail.

 

Ingreso - Olvido de Clave

  • Funcionalidades Banconexión 2.0 /
  • Ingreso - Olvido de Clave

Si el usuario olvidó su clave, entonces, cuando presione el hipervínculo “Olvido de Clave” en la pantalla de Login, se le desplegará una página donde podrá crear una nueva clave dependiendo de si tiene o no tarjeta de débito.

Olvido de Clave CON producto

Si el usuario olvidó su clave de ingreso y tiene tarjeta de débito, el sistema le solicitará: rut, clave cajero automático y la clave de su dispositivo de seguridad para la validación de la operación.

Solicitud de Clave SIN Bcx

Si el usuario olvidó su clave de ingreso y no cuenta con tarjeta de débito, el sistema le solicitará: rut, teléfono celular y mail.

 

Ingreso - Bloqueo y Desbloqueo de Clave

  • Funcionalidades Banconexión 2.0 /
  • Ingreso - Bloqueo y Desbloqueo de Clave

Si el usuario ingresó su clave 3 veces de manera errónea, entonces se bloqueará. En el caso que el usuario recuerde su clave y desee activarla, el sistema le da la opción “Reactivación Clave Bloqueada”. Si no recuerda su clave, el usuario podrá desbloquearla vía la opción de Olvido de Clave e ingresar una nueva clave.

 

Ingreso - Enrolamiento o First Login

  • Funcionalidades Banconexión 2.0 /
  • Ingreso - Enrolamiento o First Login

Con el propósito de obtener y mejorar la contactabilidad de nuestros clientes, se le solicitará al usuario pasar por un pequeño proceso de enrolamiento en el primer login y cada vez que tenga un dispositivo de seguridad pendiente de activar.

Enrolamiento CON Digicard o DigiPass Activados

Si el usuario tuvo un primer login exitoso y cuenta con dispositivo/s de seguridad activado/s, entonces se desplegará una pantalla sólo con los campos de ingreso de Celular y eMail para poder operar en la Plataforma.

Enrolamiento SIN Digicard o DigiPass Activados

Si el usuario tuvo un primer login exitoso y/o tiene un dispositivo de seguridad pendiente de activar, entonces se desplegará una pantalla con los campos de ingreso de Celular, eMail y además clave del dispositivo de seguridad para activarlo y poder operar en la Plataforma realizando transacciones.
