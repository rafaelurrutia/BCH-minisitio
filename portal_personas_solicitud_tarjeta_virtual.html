Solicitud Tarjeta Virtual - Generalidades

  • Funcionalidades Portal Personas /
  • Solicitud Tarjeta Virtual - Generalidades

Para las compras por internet, la creación de tarjetas virtuales como medio de seguridad, es una opción que cada vez más personas están realizando. En el Nuevo Portal Personas se le entrega la posibilidad al cliente, de crear, revisar y eliminar en pocos pases y de una manera fluida y eficiente, disminuyendo el tiempo requerido. Para ingresar a esta sección, el usuario tiene que hacer clic en el menú Productos > Tarjeta de Crédito > Solicitar Tarjeta Virtual.

  • Solicitud Tarjeta Virtual - Generalidades
  • Solicitar Tarjeta Virtual
  • Consultar Tarjeta
  • Eliminar Tarjeta Virtual



 
 

Al ingresar a esta sección, en el caso que el cliente posea más de una Tarjeta de Crédito, se abrirá un lightbox desde el cual tendrá que seleccionar una para Crear, Consultar o Anular una Tarjeta Virtual. Dentro de la página detalle, se mostrará un selector desde el que el cliente puede cambiar su actual selección de Tarjeta de Crédito para Crear, Consultar o Anular una Tarjeta Virtual.
