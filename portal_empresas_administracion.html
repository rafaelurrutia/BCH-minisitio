Administración - Generalidades

  • Funcionalidades Banconexión 2.0 /
  • Administración - Generalidades

Esta nueva función le permite al cliente autoadministrar los usuarios de una empresa por internet, sin tener que notificar al banco, a través del ejecutivo. Desde ella, el cliente podrá buscar, editar, desvincular, autorizar, bloquear/desbloquear y crear usuarios.

Detalle Aplicaciones

Para acceder a esta sección, se debe seleccionar el botón Aplicaciones, desde el menú que se despliega en la tabla, tras hacer clic en el ícono Tuerca. Esta ventana muestra las aplicaciones y roles que el usuario tiene asignado en la empresa seleccionada. Si desea agregar una nueva aplicación, debe presionar el botón Agregar Aplicaciones.

 
Bloqueo Usuarios

Para ingresar a esta ventana, se debe hacer clic en el botón Bloquear, desde el menú que se despliega en la tabla, tras presionar el ícono Tuerca.

Administración - Agregar Usuario

  • Funcionalidades Banconexión 2.0 /
  • Administración - Agregar Usuario

Para vincular un nuevo usuario a la empresa, se debe hacer clic en el botón Agregar Usuario. En Configuración de aplicaciones (punto 2), el cliente puede optar por agregar las aplicaciones y los roles que el usuario tendrá asignados o también puede copiar las aplicaciones y roles de un usuario existente.

Administración - Crear Nuevo Usuario

  • Funcionalidades Banconexión 2.0 /
  • Administración - Crear Nuevo Usuario

Esta sección se ejecuta de la misma forma que la opción Agregar Usuario. Para crear un nuevo usuario, el cliente debe dirigirse al Menú principal y seleccionar Administración > Usuarios > Consultar > Consultar Usuarios. En Configuración de aplicaciones (punto 2), el cliente puede optar por agregar las aplicaciones y los roles que el usuario tendrá asignados o también puede copiar las aplicaciones y roles de un usuario existente.
