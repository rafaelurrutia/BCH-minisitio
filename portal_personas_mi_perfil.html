Mi Perfil - Generalidades

  • Funcionalidades Portal Personas /
  • Mi Perfil - Generalidades

La sección “Mi Perfil” se creó para que el cliente tenga un espacio único, seguro, sencillo y personalizado donde poder administrar tanto sus datos de contacto (personales y comerciales) como sus claves y dispositivos de seguridad. En esta funcionalidad, los procesos para actualizar datos y administrar dispositivos se presentan de manera más simple y eficiente.

Esta nueva sección otorga mayor seguridad al cliente con respecto a las actuales funcionalidades de Banco en Línea, dado que exige la clave del dispositivo de seguridad tanto para visualizar como para editar sus datos y claves.

Esta funcionalidad tiene la misma lógica tanto para el portal Banco de Chile como para el Edwards Citi, solo hay diferencias de look, colores e imágenes.

Para acceder a la sección para Administrar Datos Personales y Comerciales, el usuario debe hacer click en el Menú Principal/Mi Perfil/Modificar Mis Datos/Datos Personales y de Seguridad o sobre su nombre en el Header y luego en “Editar Perfil”. Al hacer click ahí, se desplegará la siguiente pantalla.

CARACTERÍSTICAS

Los datos sensibles de contactabilidad se enmascaran, solicitando un dispositivo de seguridad para poder acceder y editar la información.

Se agregan nuevas herramientas para personalizar el perfil, permitiendo que el usuario ingrese una imagen que lo identificará a lo largo de la plataforma y los procesos que realice.

Si el cliente no tiene dispositivo de seguridad asociado, el sistema no le permite acceder a ver o editar sus datos.
